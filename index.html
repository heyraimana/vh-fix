vh-fix

An easy to use library that makes vh units work in iOS Chrome

vh-fix makes elements with vh units render more consistently in iOS Chrome. It's painless, tiny, and aims to be an easy choice for normalizing vh behavior. You can read more about vh-fix and the source code is available on GitHub: raimanadh/vh-fix.

Download v1.0.0 85vh

Usage

  1. Download vh-fix
  2. Insert <script src="path/to/vh-fix.js"></script>
  3. Add class="vh-fix" to elements using vh units
